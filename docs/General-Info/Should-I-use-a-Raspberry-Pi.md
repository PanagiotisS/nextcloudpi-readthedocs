## 嗨, 我正在使用 NextCloudPi，Raspberry Pi 對我來說是最好的選擇嗎？

這是當然的！我們的系統就是針對 Raspberry 所設計的。但是基於系統規格，這裡有些[缺點](https://ownyourbits.com/2019/02/02/whats-wrong-with-the-raspberry-pi/)。

- Pi 3b+ 是千兆網絡的不錯選擇，但有更好的選擇！
- Pi 2 工作正常。
- Pi 3 的工作效果略好於2型號。
- Pi Zero W 的性能非常低。
